# Getting Started

https://github.com/elsangedy/use-input-mask

https://github.com/text-mask/text-mask

## Install

```bash
yarn add use-input-mask
```

## Hook

```js
import useInputMask from 'use-input-mask'

const onChange = useInputMask({
  input,
  onChange,
  guide,
  keepCharPositions,
  mask,
  pipe,
  placeholderChar,
  showMask,
  initialValue,
})
```
